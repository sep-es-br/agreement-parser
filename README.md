# agreement-parser

Plugin Pentaho responsável por fornecer Contratos e Convênios ao OpenPMO por meio dos contratos definidos no `agreement-core`.

As credenciais podem ser informadas pelas variáveis de ambiente `PENTAHO_BI_USER_ID` e `PENTAHO_BI_PASSWORD`. Quando o plugin for carregado pelo OpenPMO, também são aceitas as propriedades existentes `pentahoBI.userId` e `pentahoBI.password`.
